# Changelog

Tous les changements notables de ce projet seront documentés dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/),
et ce projet adhère à [Semantic Versioning](https://semver.org/lang/fr/).

## [1.0.0] - 11-12-2025

### Ajouté
- Support Bitcoin (testnet et mainnet)
- Support Ethereum (testnet et mainnet)
- Génération d'adresses de paiement
- Suivi des confirmations blockchain
- Intégration CoinGecko pour les taux de change
- Système de webhooks avec retry et signature HMAC
- Chiffrement des données sensibles (AES-256)
- Audit trail complet
- API REST complète
- Jobs planifiés pour vérification automatique
- Commandes Artisan pour génération d'adresses
- Documentation complète

### Sécurité
- Chiffrement des données sensibles
- Webhooks signés avec HMAC-SHA256
- Validation stricte des adresses
- Protection contre les injections
